Task-2: Implement the subroutine to find the minimum st-cut of a network flow graph.

 

Algorithm:

                       

Minimum st-cut of a Network flow graph:

Idea here is simple.

·       Perform Ford-Fulkerson algorithm.

·       After the loop ends we get final residual graph.

·       Now perform DFS/BFS to mark true all reachable vertices.

·       And finally print all edges from reachable to non-reachable vertex in original graph.

 

Time Complexity:    O(max-flow*E) where E is No.of Edges.

Adjacency matrix Implementation :  O(EV3)
